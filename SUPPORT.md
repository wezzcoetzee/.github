# Support

## Getting Help

- **Questions & discussions** — use [GitHub Discussions](../../discussions)
- **Bug reports** — open a [Bug Report issue](../../issues/new?template=bug_report.yml)
- **Feature requests** — open a [Feature Request issue](../../issues/new?template=feature_request.yml)

## Please Don't Use Issues For

- General "how do I…" questions (use Discussions instead)
- Reporting security vulnerabilities (see [SECURITY.md](SECURITY.md))

## Response Times

This is a personal project maintained in spare time. Responses are best-effort and not guaranteed within any specific timeframe.
